### Виконання роботи
* Результати виконання завдань:
    1. Розроблено клас `CardGame`, який моделює карткову гру
    2. Кожна карта має атрибути: `name` (назва), `attack` (атака), `health` (здоров'я), які генеруються випадково
    3. Реалізовано метод `hit_another_card()` для атаки однієї карти іншою
    4. Застосовано **статичний приватний метод** `__calculate_critical_damage()` для розрахунку критичного удару:
        - 20% шанс критичного удару (подвійні пошкодження)
        - 10% шанс ухилення (половина пошкоджень)
        - 5% шанс промаху (нульові пошкодження)
    5. Реалізовано симуляцію 5 ходів гри між двома картами
    6. Використано `random.shuffle()` для випадкового визначення черговості атаки

* Структура файлів:
    - [game.py](game.py) — основний файл з класом `CardGame`

* Приклад оголошення класу:
```python
class CardGame:
    CARD_NAMES = ["Flame Dragon", "Ice Golem", "Thunder Phoenix", ...]
    def __init__(self):
        self.name = random.choice(CardGame.CARD_NAMES)
        self.attack = random.randint(1, 10)
        self.health = random.randint(20, 40)
```
