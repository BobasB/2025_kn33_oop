# Звіт до роботи
## Тема: _Основні парадигми ООП_
### Мета роботи: _Ознайомитись з ключовими поняттями об'єктно-орієнтованого програмування (ООП) у Python та навчитися реалізовувати їх у власних класах на прикладі ігрової симуляції._

---
### Виконання роботи
* Результати виконання завдань:
    1. Вивчено та застосовано чотири ключові парадигми ООП:
        - **Інкапсуляція** — приховання внутрішнього стану об'єктів через атрибути та методи
        - **Наслідування** — створення нових класів на основі існуючих
        - **Поліморфізм** — можливість використовувати об'єкти різних класів через єдиний інтерфейс
        - **Абстракція** — визначення загальних інтерфейсів без конкретної реалізації
    2. Розроблено ієрархію класів для ігрової симуляції
    3. Реалізовано механізм перевизначення методів (`override`)
    4. Застосовано абстрактні класи та методи (`ABC`, `abstractmethod`)
    5. Використано магічні методи Python (`__str__`, `__repr__`, `__len__`)

* Структура файлів:
    - [note.ipynb](note.ipynb) — ноутбук з теорією та прикладами виконання

- Отримано навички проектування ієрархії класів
